# Joke_SkillEditor

#### 介绍
Joker---ARPG系列课（三）：技能编辑器动画篇(已完结)

Joker---ARPG系列课（四）：技能音效与特效篇（持续更新中）

第四季主要内容：
1.多轨道样式，轨道可以新增、删除、命名等
2.技能编辑器音效轨道，运行时的音效逻辑（涉及到和当前框架音效系统的配合问题）
3.技能编辑器特效轨道（编辑器下支持预览），运行时的逻辑
